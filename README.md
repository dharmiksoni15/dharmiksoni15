## Featured Projects

### 🧠 CareerBoost AI — AI-Powered Resume Analyzer
Full stack platform that scores resume-to-job-description compatibility using the Google Gemini API with structured JSON prompting.

**Key Features:** AI-driven JD–resume compatibility scoring · strengths/gaps/suggestions generation · secure PDF upload pipeline (Multer + pdf-parse) · JWT auth with bcrypt and protected routes

**Tech Stack:** React · Node.js · Express · MongoDB · Gemini API · JWT · Tailwind CSS

🔗 [Live Demo](https://career-boost-ai-xi.vercel.app) · [GitHub](https://github.com/dharmiksoni15)

### ✅ TaskFlow — MERN Stack Task Manager
Task management application with full CRUD, live search/filtering, inline editing, and an analytics summary.

**Key Features:** Real-time UI updates · search and filter by status · inline editing with toast notifications · MVC architecture with Express and Mongoose

**Tech Stack:** React · Node.js · Express · MongoDB · Tailwind CSS · Axios

🔗 [Live Demo](https://task-manager-virid-kappa.vercel.app) · [GitHub](https://github.com/dharmiksoni15)

### 🖥️ MERN Stack Admin Dashboard
Full stack admin panel with JWT authentication, role-based authorization, and server-side validation.

**Key Features:** JWT + role-based admin authorization · full CRUD for users and contacts · Zod schema validation · React Context API for global state · confirmation prompts on destructive actions

**Tech Stack:** React (Vite) · Node.js · Express · MongoDB · JWT · bcrypt · Zod · Tailwind CSS

🔗 [Live Demo](https://mern-stack-project-three-flame.vercel.app) · [GitHub](https://github.com/dharmiksoni15)
