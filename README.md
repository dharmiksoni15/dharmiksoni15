### 🧠 [CareerBoost AI](https://github.com/dharmiksoni15/careerboost-ai) — AI Resume Analyzer

```
Resume × Job Description → AI Match Score → Actionable Insights
```

<details>
<summary><b>View Architecture & Features</b></summary>
<br/>

| Layer | Tech |
|---|---|
| Frontend | React.js + Tailwind CSS + Axios |
| Backend | Node.js + Express.js + MVC |
| AI Engine | Google Gemini API (Structured JSON Prompting) |
| File Processing | Multer + pdf-parse |
| Auth | JWT + bcrypt + Protected Routes |
| Database | MongoDB Atlas |
| Deploy | Vercel (FE) + Render (BE) |

**What it does:**
- 📥 Accepts PDF resumes via Multer upload pipeline
- 🔍 Extracts text with `pdf-parse` and sends to Gemini API
- 🎯 Returns match score, skill gaps, strengths & suggestions
- 🔐 Secure multi-user access with JWT auth

</details>

### 🛡️ [MERN Admin Dashboard](https://github.com/dharmiksoni15/Mern_stack_Project) — Role-Based Auth System

```
Register → Login → Role Check → Admin Panel → CRUD
```

<details>
<summary><b>View Architecture & Features</b></summary>
<br/>

| Layer | Tech |
|---|---|
| Frontend | React.js (Vite) + Context API + Tailwind |
| Auth | JWT + bcrypt + RBAC |
| Validation | Zod (server-side schema) |
| Database | MongoDB Atlas |
| Deploy | Vercel + Render |

**What it does:**
- 🔐 JWT authentication + bcrypt password hashing
- 👮 Role-based access — admin sees admin panel, users don't
- 📋 Full CRUD: user management + contact form handling
- ✅ Zod schema validation on every API endpoint

</details>
